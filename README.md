# MenuLens

**Experience food before you order.**

MenuLens is a mobile-first web application that helps users visualize restaurant menus with high-quality "real" photos, nutritional information, and social proof.

## Features

- **Visual Menu**: See what dishes actually look like.
- **Nutrition Facts**: Detailed calorie and nutrient breakdown.
- **Social Proof**: Real reviews and photos from other diners.
- **Premium Design**: Gold & Black aesthetic (iPhone 15 Optimized).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
